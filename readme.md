# My vim configuration.

## installation

    $ make

if you have .vimrc on your $HOME, `MakeFile` leave `.orig` files.

 - [Auto complete popup][acp]
 - [Auto close][autoclose]
 - [gundo][gundo]
 - [FuzzyFinder][FuzzyFinder]

 - [fugitive][fugitive]
 - [powerline][powerline]
